# BUDGET CALCULATOR
## PROBLEM STATEMENT

For a lot of people, managing monthly finances can be difficult. Users often have trouble keeping track of income, expenses, and balance, which can lead to poor management of their budget. There is a demonstrated need for a simplistic tool that helps users to easily document their expenses, compute total spending, and see their financial picture.

## SCOPE OF THE PROJECT

Users will be able to input their monthly income and record multiple expenses into the system.

The system will calculate:

Total expenses

Remaining balance

The system will display the expense list and summary clearly and understandably.

This project is focused on a Python application, using command-line only; therefore, advanced analytics, data storage, or graphical user interfaces are not included in this version.

## TARGET USERS

Individuals seeking to keep a record of pocket money and/or monthly allowance. 

Working professionals tracking their monthly expenses.

Any person wanting a simple budget plan without where to use complex encryption and encryption.

## HIGH-LEVEL FEATURE

Enter your monthly income.

Input and save multiple expenses by name with amounts.

Automatic calculations of:

Total amount spent
Total amount remaining
Display all expenses in an organized format.
No errors with numeric values for expenses.

Beginner programmers who want to learn the most basic input and dictionary functionality in Python.
